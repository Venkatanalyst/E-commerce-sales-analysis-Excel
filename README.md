# E-commerce Sales Performance Analysis 📊

## 📋 Project Overview
This project analyzes an E-commerce sales dataset using Microsoft Excel.
The goal of the analysis is to clean the dataset, explore customer behavior, and generate meaningful business insights using Excel tools such as Pivot Tables, formulas, and data visualization.

The project demonstrates fundamental data analyst skills including data cleaning, data exploration, and business insight generation.

## 🛠️ Tools Used
* **Microsoft Excel:**
    * **Pivot Tables:** For data aggregation and summarization.
    * **Data Cleaning:** Handling missing values and formatting.
    * **Data Analysis Toolpak:** For descriptive statistics.
    * **Visualization:** Interactive Dashboards using Slicers, Column Charts, and Area Charts.

## 📁 Dataset Description
The analysis is based on a multi-sheet dataset containing:
* **Customer Data:** Demographics (Age, Gender, Location) and Loyalty levels.
* **Product Data:** Category, Sub-category, Brand, Cost, and Stock levels.
* **Store Data:** Store type (Flagship, Online, Outlet) and Regional locations.
* **Sales Overall:** Transactional records including Order Date, Revenue, Quantity, and Payment Methods.

## 🔍 Key Analysis & Insights

### 1. Category Performance
* **Top Performer:** The **Sports** category is the leader both in volume (129 units) and revenue ($149,718).
* **Lowest Volume:** **Clothing** represents the lowest sales volume with 68 units.

### 2. Regional & Geographic Trends
* **North Region dominance:** The North region is the primary revenue driver, contributing over **57% of total revenue** ($327,336).
* **Top States:** Vermont, Connecticut, and Kansas are among the highest-performing states by quantity sold.

### 3. Customer Spending Behavior
* **Gender Insights:** Females have a higher average spend per transaction (**$1,232.51**) compared to Males (**$1,065.76**).
* **Regional Peak:** Females in the **South** region lead the entire dataset with the highest average revenue per order ($1,469.84).

### 4. Seasonal & Payment Trends
* **Peak Month:** **September** is the highest revenue month ($57,511).
* **Payment Preference:** PayPal and Credit Cards are neck-and-neck as the most preferred payment methods, while Cash on Delivery (COD) saw a unique surge in late summer (August/September).

## 🚀 Outcome & Recommendations
* **Inventory Management:** Increase stock for the Sports and Electronics categories before the July-September peak season.
* **Marketing Focus:** Develop targeted campaigns for the North region to maintain dominance and create "High-Value" loyalty programs for female customers in the South.
* **Payment Optimization:** Since PayPal and Credit Cards are preferred, ensure seamless integration and perhaps offer incentives for COD users to switch to digital for faster reconciliation.
  
## 🏁 Conclusion & Final Thoughts
This project demonstrates how Microsoft Excel can be used for effective data analysis and visualization. It highlights important data analyst skills such as data cleaning, data exploration, and insight generation.
